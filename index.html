Research Highlights

A snapshot of the questions I'm exploring and the principles that guide my approach to embodied intelligence.

Interactive Task Planning

Designing hierarchical policies that translate open-ended language into grounded action plans for dexterous manipulation.

Generalization & Transfer

Building reusable skills that adapt across environments by aligning perception, action, and semantics in shared latent spaces.

Reliable Autonomy

Selected Projects

From lab prototypes to field-tested systems, here are a few initiatives that capture the breadth of my work.

Long Horizon Hierarchy

A hierarchical framework that composes vision-language models with reinforcement learning to execute multi-step manipulation tasks reliably.

  • Multi-stage task decomposition with language abstractions
  • Self-evaluation modules for safe policy switching
  • Deployed on Franka and UR5 platforms
Project Page Code

Assistive Telepresence

Co-designed a mobile manipulator with clinicians to deliver remote care, enabling safe human-robot handoffs through shared autonomy.

  • Hybrid autonomy stack blending teleop and autonomy
  • Multimodal interface for caregivers and patients
  • User studies with rehabilitation partners

Skill Libraries for Industry 4.0

Developed transferable skill primitives for manufacturing cells, reducing deployment time for new assembly tasks by over 40%.

  • Parameterized skill representations for rapid reuse
  • Learning-from-demonstration with safety envelopes
  • Validated with collaborative robot arms
